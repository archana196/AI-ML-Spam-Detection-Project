# AI-ML-Spam-Detection-Project
# SMS Spam Detection using Machine Learning

This project builds a supervised machine learning model to classify SMS messages as spam or ham.

Technologies Used:
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn
- Jupyter Notebook

Models Used:
- Logistic Regression
- Random Forest

Evaluation Metrics:
- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC
